Bridge Addons plugin extends your Beaver Builder plugin with advanced modules.

== Description ==

Extend your Beaver Builder with advanced modules for free.


* [Button](https://bridgeaddons.com/modules/button/) - Create beautiful buttons using our button module.
* [Heading](https://bridgeaddons.com/modules/heading/) - Style your Headings with Headings module.
* [Icon List](https://bridgeaddons.com/modules/icon-list/) - Add some style to your lists with icons.
* [Image](https://bridgeaddons.com/modules/image/) - Shape your images with our image module.
* [Info Box](https://bridgeaddons.com/modules/info-box/) - Display your services with info box.
* [Notification](https://bridgeaddons.com/modules/notification/) - Use notification module to inform your users.
* [Pricing Table](https://bridgeaddons.com/modules/pricing-table/) - Display your prices using our price table module.
* [Team Member](https://bridgeaddons.com/modules/team-member/) - Introduce your team members using our team member module.
* [Testimonial](https://bridgeaddons.com/modules/testimonial/) - Display what other say about your work.
* [Testimonial Slider](https://bridgeaddons.com/modules/testimonial-slider/) - Display what other say about your work in slider.

== Installation ==

1) Install Bridge Addons either via the WordPress plugin directory or by uploading the files to your server at wp-content/plugins.

2) Activate Bridge Addons plugin in your dashboard


== Frequently Asked Questions ==

= Installation Instructions =

1) Install Bridge Addons either via the WordPress plugin directory or by uploading the files to your server at wp-content/plugins.

2) Activate Bridge Addons plugin in your dashboard.

= What are the requirements to use the Bridge Addons? =

Bridge Addons work well with all themes. Latest version of Beaver Builder pluign is required on your site in order for Bridge Addons to work.

= Can I use Bridge Addons on client sites? =

Yes you can.

= Do I need coding experience to use Bridge Addons? =

Not at all. Bridge Addons are drag and drop modules. No coding expaience required.
